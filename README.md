# Complex Number Class in C++

A simple C++ program that implements a **Complex Number** class using **Object-Oriented Programming (OOP)** and **operator overloading**.

## Features
- Addition, subtraction, multiplication, and division
- Operator overloading (`+`, `-`, `*`, `/`, `=`, `<<`)
- Constructor with default arguments
- Getter functions

## Concepts Used
- Classes & Objects
- Constructors
- Operator Overloading
- Friend Functions
- Encapsulation

## Run


g++ main.cpp -o complex_number
./complex_number
